	Razor is a virtual assistant made using python supervised machine learnig method,it works on commands of the user for specific commands with which the machine is trained.
	It is developed and tested on windows python 3.8 version.
	It answers the following questions:
1.open youtube
2.open google
3.which day it is?
4.tell me your name
5.bye
6.from wikipedia etc.

	You can train the module for various activites according to your specifications.
